# QT project base

---

This is just base for QT projects

# Installation:

cd project root and run: 
```cmd
    vcpkg_install/install.?
    vcpkg/vcpkg install qt5:PROC-OS

    vcpkg/vcpkg integrate install

    cmake build -B build -DCMAKE_TOOLCHAIN_FILE=${***previous output***}
    cmake --build build
```
